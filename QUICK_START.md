# 🚀 Quick Start Guide - Portfolio & Alexandria Dock

## Démarrage Rapide

```bash
# Méthode 1 : Via le script
cd ~/Second-Me
./start-portfolio.sh

# Méthode 2 : Via le lien symbolique
cd ~/Second-Me/portfolio-app
pnpm dev

# Méthode 3 : Chemin complet
cd ~/alexandria/anima-mundi/frontend/v0-integrations-page
pnpm dev
```

## 📍 URLs

Une fois le serveur démarré, ouvrez votre navigateur :

- **Page d'accueil** : http://localhost:3000/
- **Portfolio** : http://localhost:3000/portfolio
- **Alexandria Dock** : http://localhost:3000/alexandria-dock
- **Intégrations** : http://localhost:3000/integrations

## 📚 Documentation Complète

- `PORTFOLIO_README.md` - Guide détaillé d'utilisation
- `INTEGRATION_SUMMARY.md` - Résumé des modifications effectuées

## ⚡ Commandes Utiles

```bash
# Installer les dépendances
pnpm install

# Développement (port 3000)
pnpm dev

# Build de production
pnpm build

# Lancer la version de production
pnpm start

# Linter
pnpm lint
```
